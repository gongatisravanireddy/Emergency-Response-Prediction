# Predictive Analytics for Emergency Response Optimization Using Historical Incident Data

## Project Overview
This project uses Machine Learning techniques to analyze historical emergency incident data and predict incident severity. The system helps identify risk patterns and supports faster emergency response planning.

## Features
- Data Analysis and Visualization
- Incident Severity Prediction
- Risk Assessment
- Machine Learning Model Training
- Interactive Dashboard

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Streamlit

## Project Files
- app.py : Main application file
- model.py : Machine Learning model training file
- dataset.csv : Historical incident dataset

## How to Run

1. Install required libraries:

```bash
pip install pandas numpy scikit-learn matplotlib streamlit
```

2. Run the model:

```bash
python model.py
```

3. Start the application:

```bash
streamlit run app.py
```
